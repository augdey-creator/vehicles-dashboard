# Vehicles Dashboard

Aplicación web desarrollada con Streamlit para analizar datos de anuncios de venta de autos.

## Funcionalidad
- Visualización de histogramas
- Gráficos de dispersión
- Interfaz interactiva

## Tecnologías
- Python
- Pandas
- Plotly Express
- Streamlit
